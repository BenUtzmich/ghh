This package includes these files:
	
	CreateDatabase.sql
	index.php
	GHH - Honeynet.jpg

index.php is used to view logs. You will need to edit the index.php configuration
section to connect to the database, and set a login password.

CreateDatabase.sql contains the SQL to create the GHH log table structure.

GHH - Honeynet.jpg is a quick picture of a typical GHH honeynet.
